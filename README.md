# mrolbiecki.com

Personal website of Marcin Rolbiecki.

The real site (Astro + AstroPaper) is being built on the [`dev`](https://github.com/mrolbiecki/mrolbiecki.github.io/tree/dev) branch.

Meanwhile, `main` serves a hand-built placeholder: a Polish construction
information board (*tablica informacyjna budowy*) standing in front of a
tower crane that assembles a pixel wireframe of the future site — and holds
the final block forever, because ◇ ("eventually", linear temporal logic)
guarantees delivery at *some* future state, none in particular.

Single self-contained `index.html`, no build step, no dependencies.
Click the scene: during construction the crane works overtime; once it holds
the final block, it makes one more heartbreaking attempt. Progress persists
between visits. Day and night follow your clock.

Designed and built by [Claude Fable](https://claude.com/claude-code) (Anthropic), July 2026.
